MariaDB 설정 가이드

설정 파일 경로

MariaDB의 설정 파일은 다음 경로에 위치합니다:

/etc/mysql/mariadb.conf.d/50-server.cnf

외부 통신 설정

MariaDB가 외부에서 접속 가능하도록 하려면 다음 설정을 수정해야 합니다:

# bind-address = 127.0.0.1

위 라인을 주석 처리하거나 다음과 같이 수정합니다:

bind-address = 0.0.0.0

이렇게 설정하면 MariaDB가 모든 IP 주소에서 접속 가능하게 됩니다.

포트 설정

MariaDB의 기본 포트는 3306입니다. 필요한 경우 다음과 같이 포트를 변경할 수 있습니다:

port = 3306

소켓 설정

MariaDB의 소켓 파일 경로를 설정합니다:

socket = /var/run/mysqld/mysqld.sock

또는

socket = /run/mysqld/mysqld.sock

소켓과 PID 파일 경로 일치

중요: MariaDB의 소켓 파일 경로와 PID 파일 경로는 일치해야 합니다. 그렇지 않으면 MariaDB가 정상적으로 작동하지 않을 수 있습니다.

PID 파일 경로를 설정하려면 다음과 같이 수정합니다:

pid-file = /var/run/mysqld/mysqld.pid

또는

pid-file = /run/mysqld/mysqld.pid

설정 적용

설정 파일을 수정한 후에는 MariaDB 서비스를 재시작하여 변경 사항을 적용해야 합니다:

service mysql restart
