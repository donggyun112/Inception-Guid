wp-config.php 설정값 설명

경로: /var/www/html/wp-config-sample.php

1. DB_NAME

define('DB_NAME', 'database_name_here');

이 설정값은 WordPress가 사용할 데이터베이스 이름을 지정합니다.

2. DB_USER

define('DB_USER', 'username_here');

이 설정값은 데이터베이스에 접속할 때 사용할 MySQL 사용자 이름을 지정합니다.

3. DB_PASSWORD

define('DB_PASSWORD', 'password_here');

이 설정값은 데이터베이스에 접속할 때 사용할 MySQL 사용자 비밀번호를 지정합니다.

4. DB_HOST

define('DB_HOST', 'localhost');

이 설정값은 MySQL 서버의 호스트 이름을 지정합니다. 로컬 서버에서 실행 중인 경우 'localhost'로 지정합니다.

5. DB_CHARSET

define('DB_CHARSET', 'utf8');

이 설정값은 데이터베이스 테이블 생성 시 사용할 문자 인코딩을 지정합니다.

6. DB_COLLATE

define('DB_COLLATE', '');

이 설정값은 데이터베이스 테이블 생성 시 사용할 정렬 순서를 지정합니다. 일반적으로 비워두는 것이 좋습니다.
