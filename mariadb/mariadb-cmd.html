mysql -e 옵션 설명

mysql -e 옵션은 MySQL 또는 MariaDB에서 단일 쿼리를 실행하고 결과를 출력하는 데 사용됩니다. 이 옵션을 사용하면 SQL 쿼리를 명령줄에서 직접 실행할 수 있습니다.

사용 방법:

mysql -u [사용자명] -p[비밀번호] -e [SQL 쿼리]

예시:

mysql -u root -p1234 -e "SELECT * FROM users"
mysql -u admin -ppassword -e "CREATE DATABASE mydatabase"
mysql -u john -psecret -e "INSERT INTO employees (name, age) VALUES ('John Doe', 30)"

위의 예시에서는 각각 users 테이블의 모든 데이터 조회, mydatabase 데이터베이스 생성, employees 테이블에 새로운 레코드 삽입을 수행합니다.

-e 옵션을 사용하면 MySQL 클라이언트를 실행하고 쿼리를 입력하는 과정을 간소화할 수 있습니다. 이는 자동화 스크립트나 일회성 쿼리를 실행할 때 유용합니다.
