Nginx를 Debian에서 Docker로 빌드 및 실행

1. Dockerfile 생성

먼저, Nginx를 빌드하기 위한 Dockerfile을 생성합니다.

FROM debian:latest

RUN apt-get update && apt-get install -y nginx

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]

2. Docker 이미지 빌드

Dockerfile이 있는 디렉토리에서 다음 명령어를 실행하여 Docker 이미지를 빌드합니다.

docker build -t my-nginx .

3. Docker 컨테이너 실행

빌드된 이미지를 기반으로 Docker 컨테이너를 실행합니다.

docker run -d -p 80:80 --name my-nginx-container my-nginx

4. 브라우저에서 접속

웹 브라우저에서 http://localhost에 접속하여 Nginx 기본 페이지가 표시되는지 확인합니다.

5. 컨테이너 중지 및 제거

실행 중인 컨테이너를 중지하고 제거하려면 다음 명령어를 사용합니다.

docker stop my-nginx-container
docker rm my-nginx-container

위의 과정을 통해 Nginx를 Debian 기반의 Docker 이미지에서 빌드하고 실행할 수 있습니다.
